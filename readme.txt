Project Sederhana "Aplikasi Report Data Karyawan"

"Report Data Karyawan" adalah sebuah project sederhana yang bertujuan untuk menampilkan data-data karyawan. Dalam project ini, saya menggunakan operasi CRUD (Create, Read, Update, Delete) di Python. CRUD adalah singkatan dari empat operasi dasar yang diperlukan untuk mengelola data dalam sistem penyimpanan seperti basis data atau aplikasi.

Tujuan menggunakan CRUD:
1. Create (Membuat): Pada project ini, operasi Create digunakan untuk membuat atau menambahkan data baru ke dalam database.
2. Read (Membaca): Operasi Read digunakan untuk membaca atau menampilkan data dari database.
3. Update (Memperbarui): Operasi Update digunakan untuk mengubah atau mengedit data yang ada dalam database.
4. Delete (Menghapus): Operasi Delete digunakan untuk menghapus data dari database.
Dengan demikian, project ini memungkinkan kita untuk mengelola data karyawan secara efektif menggunakan operasi CRUD di Python.
